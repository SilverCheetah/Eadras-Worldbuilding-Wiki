# Sealed Folder — Author Notes (NOT Tome Content)

**This folder is out-of-character (OOC) author/DM notes.**

Files here are **not** part of the in-universe [[tome-of-baldaran|Tome of Baldaran]]. They are private worldbuilding notes recording **plot-spine secrets** — facts the author/GM holds back from the wiki proper to preserve campaign or story tension.

## Rules for Claude

1. **Do not wikilink into `_sealed/` from any page outside this folder.** Pages in `wiki/` (the public Tome) must remain navigable without leaking plot secrets.
2. **Do not summarize sealed contents back into public pages.** Any cross-reference must be one-way: sealed pages may link to public pages for context, but never the reverse.
3. **When updating a public page, do not betray that a sealed counterpart exists** — phrase open-ended canon as "currently unknown to mortals," "the gods have not revealed," "lost to time," etc.
4. **Only update sealed pages on explicit user direction.** Treat the sealed file as the user's plot notebook; do not invent new plot secrets unprompted.
5. **When the user wants public-facing breadcrumbs (rumors, sailor tales, sealed-chapter teases) about a sealed topic**, those go in the public page they relate to, written as fragmentary in-world hearsay — never as the answer.

## Distinction from Tome-sealed-chapter framing

The in-universe Tome contains chapters that **open at the right time** for the right reader — that is a diegetic device, used inside `wiki/` proper to signal that some lore is in-character but currently inaccessible. **This `_sealed/` folder is something else**: meta-level author canon that exists outside the Tome entirely.
