# Protocols — Subfolder Overview

**Summary**: This subfolder collects **reusable prompt-instruction documents** — standardized frameworks for systematically expanding, revising, or auditing parts of the Eadras setting. Protocols are **out-of-universe authoring tools**, not in-universe canon; they are kept structurally separate from the main wiki for the same reason `wiki/rpg/` is.

**Sources**: Created 2026-05-16 per ingest of `raw/Eadras Race Deepening Protocol (v1).txt`. See [[log]].

**Last updated**: 2026-05-16.

> *Updated 2026-05-16 (same day): second protocol added — `core-canon-and-thematic-principles.md`. The folder now holds two protocol types: **work-driving prompts** (the race-deepening framework) and **context-loading primers** (the core-canon primer prepended to other prompts). Both share the same fenced-prompt-body discipline.*

---

## What lives here

Prompt-instruction documents intended to be **copy-pasted** into a working session (this Claude Code session, ChatGPT, Grok, etc.) to drive structured worldbuilding work. Each protocol page leads with a brief usage note, then presents the prompt itself as a single fenced code block so it can be lifted in one selection without picking up wiki formatting.

Protocols are versioned in their titles (`(v1)`, `(v2)`) so superseding revisions don't silently overwrite the prior version's behavior.

## Current contents

- **[[protocols/race-deepening-protocol|Eadras Race Deepening Protocol (v1)]]** — 10-section framework for systematically deepening any race page (origins, biology, society, spirituality, material culture, economy, relations, language, subgroups, open canon). **Type:** work-driving prompt.
- **[[protocols/core-canon-and-thematic-principles|Eadras Core Canon & Thematic Principles (v2)]]** — foundational-context primer for prepending to almost every major prompt; condensed cosmology / Elder Gods / four-order magic / defensive Weave / timeline / modern powers / themes / tone / Part I in-world principles / Part II author-side craft principles. **Type:** context-loading primer. **Drift-watch**: derived from [[principles-of-eadras]] and the cosmology/timeline pages; back-propagate canon updates here. *v2 (2026-05-16 afternoon) regenerated against current canon — post-rework magic system, 2026-05-15 Crusade canon, Amunorian rename, Dragon Empire added.*

## What does not belong here

- **In-universe canonical content.** Goes on the main wiki.
- **OOC tonal references for a specific page.** Goes in that page's `ooc-block`.
- **RPG splat / play-mechanics material.** Goes in [[rpg/README|wiki/rpg/]].
- **Real-world inspirations / authorial influences.** Goes on [[inspirations]].

The test: *is this a tool for producing wiki content, or is it wiki content?* If it's a tool, it belongs here.
