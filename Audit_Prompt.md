# Eadras Worldbuilding Audit Prompt

You are auditing my unfinished Obsidian worldbuilding wiki.

The project is not complete. The modern map exists, but cultures, nations, and biomes are not fully placed. Some modern cultures/nations exist, but not all. Deep lore and history are more developed than the present-day political/cultural landscape.

Use the entire `wiki/` folder as source material.

Generate:

`generated/Eadras_Worldbuilding_Audit.md`

Rules:
- Do not invent new lore.
- Preserve exact names and spellings.
- Include source file references.
- Flag ambiguity instead of resolving it.
- If pages disagree, mark the issue as a possible contradiction.
- Treat stubs and incomplete ideas as useful signals, not failures.

Include these sections:

## 1. Project Status Summary
What parts of the world appear developed, partial, or underdeveloped?

## 2. Canon Stability Assessment
Classify major lore areas as:
- Stable canon
- Mostly stable but needs clarification
- Contradictory
- Stub / placeholder
- Missing but implied

## 3. Continuity and Consistency Issues
Check timeline, cosmology, gods, magic, geography, cultures, nations, species, technology, religion, economics, politics, and character/faction relationships.

For each issue:
- Description
- Source files
- Why it matters
- Suggested author decision

## 4. Stub Ideas and Underdeveloped Concepts
List incomplete pages, concepts, factions, cultures, places, or historical events.

For each:
- Current information
- What is missing
- Why it matters
- Development questions

## 5. Missing Load-Bearing Worldbuilding
Identify absent or thin foundations:
climate, biomes, rivers, food systems, trade routes, resources, languages, migration, ethnic boundaries, political borders, religion in daily life, class, family, law, warfare, education, medicine, technology, currency, transport, calendars, death customs, festivals, art, gender roles, magic in society.

## 6. Map-to-Culture Development Needs
Identify likely settlement zones, trade chokepoints, conflict zones, cultural crossroads, isolated regions, frontier candidates, and imperial heartland candidates.

## 7. Deep Lore to Modern World Connections
Identify ancient lore that should have modern consequences.

For each:
- Ancient/deep lore element
- Expected modern consequence
- Whether the wiki shows it
- Suggested strengthening

## 8. Historical Causality Audit
Flag unclear causes, unclear consequences, collapses without aftermath, wars without demographic/economic effects, religions without schisms, empires without administration, and disasters without migrations/refugees.

## 9. Culture/Nation Template Gaps
For each known modern culture or nation, assess:
homeland, biome, language, government, economy, religion, hierarchy, family, military, foodways, clothing, architecture, art, taboos, internal conflicts, external rivals, historical memory, magic/divinity, neighbors.

Mark each as:
Present, Partial, Missing, Contradictory, or Not applicable.

## 10. Magic/Metaphysics Integration Audit
Check costs, limits, divine rules, social consequences, magical institutions, taboos, laws, magical economy, magical warfare, medicine, communication, and travel.

## 11. Story-Readiness Assessment
Identify worldbuilding elements that suggest strong story conflict:
personal, familial, cultural, religious, political, economic, military, magical, historical, cosmic.

## 12. Questions the Author Should Answer Next
Divide into:
- Must answer before writing
- Should answer before finalizing map/cultures
- Can answer during drafting
- Optional flavor

## 13. Recommended Next Development Passes
Recommend the next 5–10 worldbuilding passes in order.

## 14. Source File Index
List source files used.
