Draconic Suffix System
After the triconsonantal root + three vowels, suffixes handle the remaining grammar and nuance. Suffixes stack in a relatively fixed order and allow dragons to convey extremely precise information.
Suffix Order (from closest to the root → farthest)

Voice / Direction
Tense / Time
Mood / Attitude
Plurality / Scale
Honor / Social Register (can override or amplify V₂)
Power / Magical Intensity (how strongly the Fire is being called upon)

Main Suffix Categories & Examples
1. Voice / Direction

-ak – Active / Direct
-ar – Passive / Received
-ath – Causative (“causes X to happen”)
-esh – Reflexive (“X does to itself”)
-un – Benefactive (“for the benefit of”)
-ir – Malefactive (“to the harm of”)

2. Tense / Time

-en – Present / Ongoing
-in – Recent Past
-an – Distant Past
-or – Near Future
-ûr – Distant / Prophetic Future
-el – Timeless / Eternal

3. Mood / Attitude

-is – Declarative / Neutral
-os – Imperative / Command
-as – Interrogative / Questioning
-us – Dubitative (“maybe / uncertain”)
-esh – Desiderative (“I wish / desire”)
-akh – Contemptuous / Scornful

4. Plurality / Scale

-at – Plural (simple)
-akhat – Many / Great host
-ûl – All / Entirety
-im – Few / Scattered
-orim – Countless / Legion

5. Honor / Social Register

-ara – Radiant / High Honor
-adru – Respectful / Guardian-level
-rakh – Eternal / Ancient respect
-nakh – Low / Disrespectful (rarely used to elders)

6. Power / Magical Intensity

-ka – Normal strength
-kha – Strong / Potent
-khâr – Overwhelming / Cataclysmic
-thûl – Subtle / Precise control
-zhar – Wild / Uncontrolled


Root Template & Vowel System
Every root follows: C₁ V₁ C₂ V₂ C₃ V₃
The three vowel positions each carry independent layers of meaning, allowing immense complexity in a single root.























Root Template & Vowel System
